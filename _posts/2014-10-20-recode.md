最近看到了一些框架，包括前端和服务器端，记录下来，有时间逐一研究一下。

*[slim]框架
    一个轻量级的PHP5框架，用来创建RESTful的Web应用，看代码:
```
<?php
require 'Slim/Slim.php';
$app = new Slim();
$app->get('/hello/:name', function ($name) {
    echo "Hello, $name!";
});
$app->run();
```
[slim]: http://www.slimframework.com/
